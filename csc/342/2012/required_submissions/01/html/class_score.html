Detailed Description

A class of 2-digit scores for 4-digit codes, for use in a guessing game that pits the computer as codebreaker against the user as codemaker.

Member Function Documentation

bool Score::isValid ( ) const

Returns true, if this Score object represents a valid score, and false otherwise.

A valid score contains exactly 2 digits, each one in the range 0..4. Also, the sum of the digits must not exceed 4, and a score of 31 is also invalid.

bool Score::operator== ( const Score &  otherScore) const

Tests whether this Score object is the same as the otherScore object, that is, whether they each contain 2 digits and those digits are the same and in the same order in both Score objects.

bool Score::operator!= ( const Score &  otherScore) const

Tests whether this Score object is not equal to the otherScore object.

Returns true if operator== returns false, and false if operator== returns true.
